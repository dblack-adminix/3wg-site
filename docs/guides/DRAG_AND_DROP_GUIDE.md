# 🎯 Руководство по перемещению блоков (Drag & Drop)

## Возможности

Теперь в админке можно не только включать/отключать блоки, но и **менять их порядок** на главной странице!

## Как использовать

### 1. Откройте админку
```
http://localhost:8080/admin
```

### 2. Войдите как админ
- Email: `admin@3wg.ru`
- Password: `admin123`

### 3. Перейдите на вкладку "Контент сайта"

### 4. Перемещайте блоки

#### Способ 1: Мышью
1. Наведите курсор на иконку **⋮⋮** (GripVertical) слева от блока
2. Зажмите левую кнопку мыши
3. Перетащите блок вверх или вниз
4. Отпустите кнопку мыши

#### Способ 2: Клавиатурой
1. Нажмите Tab, чтобы выбрать блок
2. Нажмите Space, чтобы "поднять" блок
3. Используйте стрелки ↑↓ для перемещения
4. Нажмите Space снова, чтобы "опустить" блок

### 5. Сохраните изменения
Нажмите кнопку **"Сохранить"** в правом верхнем углу

### 6. Проверьте результат
Откройте главную страницу http://localhost:8080/ и обновите (F5)

## Пример: Поставить "Тарифы" выше "VPN секции"

1. Откройте админку → "Контент сайта"
2. Найдите блок **"Тарифы"** (обычно #5)
3. Перетащите его выше блока **"VPN секция"** (#3)
4. Теперь порядок:
   - #1 Hero секция (фиксированная)
   - #2 Keenetic прошивка
   - #3 **Тарифы** ← переместили сюда
   - #4 VPN секция
   - #5 Услуги
   - ...
5. Нажмите **"Сохранить"**
6. Обновите главную страницу

## Важные моменты

### Hero секция всегда первая
Блок **"Hero секция"** (главный баннер) **нельзя перемещать** - он всегда остается первым на странице.

### Нумерация блоков
Каждый блок имеет номер (#1, #2, #3...), который показывает его позицию на странице:
- #1 - Hero секция (фиксированная)
- #2-#11 - Остальные блоки (можно перемещать)

### Визуальная обратная связь
- При перетаскивании блок становится полупрозрачным
- Курсор меняется на "grabbing" (рука с зажатым предметом)
- Другие блоки автоматически сдвигаются, освобождая место

### Отмена изменений
Если вы передумали, нажмите кнопку **"Отменить"** - все вернется к сохраненному состоянию.

## Технические детали

### Библиотека
Используется [@dnd-kit](https://dndkit.com/) - современная библиотека для drag-and-drop в React.

### Хранение порядка
Порядок блоков сохраняется в поле `block_order` в базе данных:
```json
{
  "hero_section": true,
  "keenetic_section": true,
  "vpn_section": true,
  ...
  "block_order": [
    "pricing_section",
    "keenetic_section",
    "vpn_section",
    "services_section",
    ...
  ]
}
```

### Дефолтный порядок
Если `block_order` не определен, используется дефолтный порядок:
1. Keenetic прошивка
2. VPN секция
3. Тарифы
4. Услуги
5. Оборудование
6. Инфраструктура
7. FAQ
8. Статьи
9. Telegram
10. Статус систем

## Примеры использования

### Сценарий 1: Акция на тарифы
Хотите привлечь внимание к новым тарифам? Переместите блок "Тарифы" на второе место (сразу после Hero).

### Сценарий 2: Новая статья
Опубликовали важную статью? Переместите блок "Статьи" выше, чтобы пользователи увидели его раньше.

### Сценарий 3: Сезонная акция
Временно переместите блок с акцией в топ, а после окончания акции верните обратно.

## Troubleshooting

### Блок не перемещается
- Убедитесь, что вы тянете за иконку ⋮⋮, а не за сам блок
- Попробуйте обновить страницу (F5)

### Изменения не сохраняются
- Проверьте, что нажали кнопку "Сохранить"
- Проверьте консоль браузера на ошибки (F12)
- Убедитесь, что бэкенд запущен

### Порядок не применяется на главной странице
- Обновите главную страницу (F5)
- Очистите кеш браузера (Ctrl+Shift+R)
- Проверьте, что изменения сохранились в админке

## Горячие клавиши

- **Tab** - Переключение между блоками
- **Space** - Поднять/опустить блок
- **↑↓** - Перемещение блока вверх/вниз
- **Esc** - Отменить перемещение

## Доступность

Система drag-and-drop полностью доступна для:
- Пользователей с клавиатурой
- Скринридеров
- Тач-устройств (планшеты, смартфоны)
