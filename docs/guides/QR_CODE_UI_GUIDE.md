# QR Code - Визуальная инструкция

## Где найти QR код в админке

### Шаг 1: Откройте админку
```
URL: http://localhost:8080/admin
Логин: admin@3wg.ru
Пароль: admin123
```

### Шаг 2: Перейдите в раздел "Серверы"
- В верхнем меню выберите вкладку "Серверы"
- Вы увидите список всех серверов

### Шаг 3: Выберите сервер
- Найдите нужный сервер в списке
- Нажмите кнопку "Подробнее" (или "👁 Подробнее")

### Шаг 4: Найдите пира
- Вы увидите список всех пиров (клиентов) на сервере
- Каждый пир показывает:
  - Имя (например, "Dimitriy_PC")
  - Статус (Онлайн/Офлайн)
  - Public Key
  - Allowed IPs
  - Endpoint
  - Трафик (получено/отправлено)
  - Последний handshake

### Шаг 5: Откройте детали пира
- Найдите нужного пира в списке
- Нажмите кнопку "👁" (глаз) справа от пира
- Откроется модальное окно с деталями

### Шаг 6: Прокрутите до блока "Подключение"
В модальном окне вы увидите:
1. **Основная информация** - имя, статус, ключи
2. **Статистика трафика** - получено, отправлено, всего
3. **График трафика (7 дней)** - дневная статистика
4. **Трафик по часам (24 часа)** - почасовая статистика
5. **Подключение** ← Прокрутите сюда!

### Шаг 7: Покажите QR код
В блоке "Подключение" вы увидите две колонки:

**Левая колонка - QR код:**
- Изначально показывается иконка QR кода
- Кнопка "Показать QR"
- Нажмите кнопку → QR код появится
- Нажмите "Скрыть QR" → QR код скроется

**Правая колонка - Скачивание:**
- Заголовок "Скачать конфигурацию"
- Описание: "Импортируйте конфиг в WireGuard клиент или отсканируйте QR код"
- Кнопка "📥 Скачать .conf"
- Инструкции:
  - 📱 Мобильные: Сканируйте QR
  - 💻 Десктоп: Скачайте .conf

### Шаг 8: Используйте QR код

**Для мобильных устройств:**
1. Откройте WireGuard на телефоне
2. Нажмите "+" → "Сканировать QR код"
3. Наведите камеру на QR код на экране
4. Подключение добавится автоматически

**Для десктопа:**
1. Нажмите "Скачать .conf"
2. Файл сохранится как `<имя-пира>.conf`
3. Откройте WireGuard GUI
4. Импортируйте файл
5. Подключитесь

## Цвета и стили

- **Фон модалки:** `#0a0a0a` (темный)
- **Фон блоков:** `#111` (чуть светлее)
- **Границы:** `border-border` (серый)
- **QR код фон:** `white` (белый для контраста)
- **QR код размер:** 192x192px
- **Кнопки:** Outline стиль с hover эффектом
- **Иконки:** Lucide React (QrCode, FileDown)

## Анимации

- **Загрузка:** Спиннер вращается
- **Показать/Скрыть QR:** Плавное появление/исчезновение
- **Hover на кнопках:** Изменение фона

## Адаптивность

- **Десктоп:** Две колонки (QR + Скачивание)
- **Планшет:** Две колонки (уменьшенные)
- **Мобильный:** Одна колонка (вертикально)

## Доступность

- ✅ Клавиатурная навигация
