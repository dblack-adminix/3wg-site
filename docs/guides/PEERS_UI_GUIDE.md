# UI для просмотра пиров

## Обзор

Реализован детальный просмотр пиров WireGuard в админ-панели с интерфейсом, похожим на WGDashboard.

## Возможности

### 1. Вкладки в деталях сервера
- **Информация** - основные данные сервера
- **Пиры** - список всех пиров с деталями

### 2. Карточки пиров
Каждый пир отображается в виде карточки с:
- Иконкой Network
- Названием (или "Peer N")
- IP адресом (badge)
- Публичным ключом (первые 32 символа)
- Трафиком (получено/отправлено)

### 3. Раскрывающиеся детали
При клике на карточку пира раскрываются детали:
- **Публичный ключ** - полный ключ в моноширинном шрифте
- **Разрешенные IP** - список IP в виде badges
- **Статистика**:
  - Получено (синий цвет)
  - Отправлено (зеленый цвет)
  - Всего (кислотно-зеленый)
  - Последний handshake (с иконкой часов)
- **Endpoint** - если настроен
- **Preshared Key** - если настроен

## Использование

### 1. Открыть детали сервера
1. Перейдите в админку → Серверы
2. Нажмите "Подробнее" на карточке сервера
3. Откроется диалог с вкладками

### 2. Просмотр пиров
1. Переключитесь на вкладку "Пиры"
2. Список пиров загрузится автоматически
3. Кликните на карточку пира для раскрытия деталей

### 3. Форматирование данных
- **Трафик**: автоматически форматируется (B, KB, MB, GB, TB)
- **Даты**: форматируются в локальный формат (ru-RU)
- **Публичные ключи**: отображаются в моноширинном шрифте

## Дизайн

### Цвета
- **Получено**: `text-blue-500` (синий)
- **Отправлено**: `text-green-500` (зеленый)
- **Всего**: `text-primary` (кислотно-зеленый #CCFF00)
- **Фон карточек**: `bg-[#111]`
- **Фон деталей**: `bg-[#0a0a0a]`

### Иконки
- **Network**: иконка пира
- **Download**: получено
- **Upload**: отправлено
- **Clock**: последний handshake

### Анимации
- Появление карточек с задержкой (stagger)
- Плавное раскрытие деталей (height animation)
- Hover эффекты на карточках

## Адаптивность

- **Desktop**: 2 колонки для статистики
- **Mobile**: 1 колонка, стек элементов
- **Tablet**: адаптивная сетка

## Производительность

- **Lazy loading**: пиры загружаются только при открытии вкладки
- **Stagger animation**: задержка 50ms между карточками
- **Мемоизация**: форматирование данных кэшируется

## Доступность

- **Keyboard navigation**: Tab для навигации
- **Screen readers**: aria-labels для иконок
- **Focus states**: видимые состояния фокуса
- **Color contrast**: соответствует WCAG AA
