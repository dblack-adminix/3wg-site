## Команды для выполнения

### 1. Применить миграцию
```powershell
cd backend
.\apply-migration.ps1
```

### 2. Пересобрать бэкенд
```powershell
docker-compose down
docker-compose up --build -d
```

### 3. Проверить логи
```powershell
docker logs vpn-backend -f
```

### 4. Добавить сервер
```powershell
cd ..
.\add-test-server.ps1
```

### 5. Протестировать
```powershell
.\test-wgdashboard-integration.ps1
```
