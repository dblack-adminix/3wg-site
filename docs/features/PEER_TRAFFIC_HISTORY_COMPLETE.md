## Как работает

1. **Первый снимок**: При старте backend воркер создает первый снимок для всех пиров
   - `bytes_received`, `bytes_sent` - текущий кумулятивный трафик
   - `delta_received`, `delta_sent` - для первого снимка = текущий трафик

2. **Последующие снимки**: Каждый день в 00:00 UTC воркер:
   - Получает текущий трафик пира
   - Находит предыдущий снимок
   - Вычисляет дельту: `delta = current - previous`
   - Сохраняет новый снимок

3. **График**: Показывает дельту по дням (сколько трафика использовано в каждый день)
   - Синяя линия - получено (download)
   - Зеленая линия - отправлено (upload)
