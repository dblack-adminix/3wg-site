# Статус: Отображение загрузки серверов ✅

## Что сделано

Добавлено визуальное отображение текущей загрузки серверов в админке.

### Backend

- ✅ Подсчет текущих пиров из кэша `wg_peers_cache`
- ✅ API возвращает `current_peers` для каждого сервера
- ✅ Быстрый подсчет (< 10ms) без обращения к WGDashboard

### Frontend

- ✅ Отображение "Пиры: 15/100" в карточке сервера
- ✅ Прогресс-бар с цветовой индикацией:
  - 🟢 Зеленый (0-60%) - нормальная загрузка
  - 🟡 Желтый (60-80%) - средняя загрузка
  - 🔴 Красный (80-100%) - высокая загрузка
- ✅ Предупреждение "Сервер близок к лимиту!" при >80%
- ✅ Анимированный градиент на прогресс-баре

## Как проверить

1. Открыть админку: http://localhost:5173/admin
2. Перейти на вкладку "Серверы"
3. Каждая карточка сервера показывает:
   - Текущее количество пиров
   - Максимальное количество (из `max_users`)
   - Визуальный прогресс-бар
   - Предупреждение при высокой загрузке
