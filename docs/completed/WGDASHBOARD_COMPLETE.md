## 🔧 Технические решения

### Проблема: Отсутствие эндпоинтов для пиров

**Решение:** Использование raw конфигурации
1. Получаем raw конфигурацию через `/api/getWireguardConfigurationRawFile`
2. Парсим INI формат (секции `[Peer]`)
3. Модифицируем конфигурацию
4. Обновляем через `/api/updateWireguardConfigurationRawFile`

### Проблема: GORM автомиграция с массивами PostgreSQL

**Решение:** Ручные SQL миграции
1. Отключена автомиграция GORM
2. Используется `pq.StringArray` для поля `protocols`
3. SQL миграции применяются вручную через скрипты
