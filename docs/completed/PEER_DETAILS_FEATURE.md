# Функционал деталей и удаления пиров ✅

## Дата: 29 января 2026

## Что сделано

### 1. Удаление пира ✅

**Реализовано**:
- Кнопка удаления (красная с иконкой корзины)
- Диалог подтверждения удаления
- Вызов API метода `removePeerFromServer(serverId, publicKey)`
- Обновление списка пиров после удаления
- Toast уведомления об успехе/ошибке

**Как работает**:
1. Пользователь нажимает кнопку удаления на пире
2. Открывается диалог подтверждения с именем пира
3. При подтверждении отправляется DELETE запрос к WGDashboard
4. Пир удаляется с сервера
5. Список пиров обновляется автоматически

### 2. Просмотр деталей пира ✅

**Реализовано**:
- Кнопка "Подробнее" (иконка глаза)
- Модальное окно с полной информацией о пире
- Отображение основной информации:
  - Имя пира
  - Статус (Онлайн/Офлайн)
  - Public Key
  - Allowed IPs
  - Endpoint (если есть)
- Статистика трафика:
  - Получено (Download) - синий
  - Отправлено (Upload) - зеленый
  - Всего (Total) - кислотно-зеленый
  - Последний handshake

## API

### Удаление пира

**Метод**: `api.removePeerFromServer(serverId, publicKey)`

**Эндпоинт**: `DELETE /api/v1/admin/servers/:id/wg/peers/:peer_id`

**Параметры**:
- `serverId` - ID сервера
- `publicKey` - Public key пира (используется как ID)

**Ответ**:
```json
{
  "status": "success",
  "message": "Peer removed successfully"
}
```

## Использование

### Просмотр деталей пира

1. Открыть админку → вкладка "Серверы"
2. Нажать "Подробнее" на карточке сервера
3. Выбрать конфигурацию (wg0, wg1, и т.д.)
4. Нажать кнопку 👁️ (глаз) на нужном пире
5. Откроется модальное окно с деталями

### Удаление пира

1. Открыть админку → вкладка "Серверы"
2. Нажать "Подробнее" на карточке сервера
3. Выбрать конфигурацию (wg0, wg1, и т.д.)
4. Нажать кнопку 🗑️ (корзина) на нужном пире
5. Подтвердить удаление в диалоге
6. Пир будет удален с сервера


### Функции

```typescript
// Удаление пира
const handleDeletePeer = async () => {
  if (!deletingPeer || !selectedServerId) return;
  
  try {
    await api.removePeerFromServer(selectedServerId, deletingPeer.publicKey);
    toast.success('Пир удален');
    setIsDeletePeerDialogOpen(false);
    setDeletingPeer(null);
    loadPeersForConfig(activeConfig);
  } catch (error: any) {
    console.error('Failed to delete peer:', error);
    toast.error(error.message || 'Ошибка удаления пира');
  }
};
```

### Цветовая схема

- **Онлайн**: `bg-green-500/20 text-green-500`
- **Офлайн**: `bg-gray-500/20 text-gray-500`
