- `check-my-keys.ps1` - Проверка моих ключей
- `check-peer-*.ps1` - Проверка пиров
- `check-server-*.ps1` - Проверка серверов
- `check-wgdashboard-*.ps1` - Проверка WGDashboard

- `get-auth-token.ps1` - Получение токена авторизации
- `update-*.ps1` - Скрипты обновления
