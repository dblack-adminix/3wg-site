## Как протестировать

### 1. Открыть админку
```
http://localhost:5173/admin
```

### 2. Перейти на вкладку "Серверы"

### 3. Выбрать сервер с пирами
Нажать кнопку "Подробнее" на любом сервере

### 4. Выбрать конфигурацию (wg0, wg1)
Переключиться между вкладками конфигураций

### 5. Тестирование просмотра деталей пира
1. Найти любой пир в списке
2. Нажать кнопку с иконкой глаза (Eye) "Подробнее"
3. **Ожидаемый результат:**
   - ✅ Модальное окно открывается СРАЗУ
   - ✅ Модальное окно отображается ПОВЕРХ списка пиров
   - ✅ Фон затемнен (overlay)
   - ✅ Видна вся информация о пире:
     - Основная информация (имя, статус, public key, allowed IPs, endpoint)
     - Статистика трафика (получено/отправлено/всего/handshake)
     - График трафика за 7 дней (демо данные)
     - Карта геолокации (демо координаты Москвы)

### 6. Тестирование удаления пира
1. Найти любой пир в списке
2. Нажать кнопку с иконкой корзины (Trash) красного цвета
3. **Ожидаемый результат:**
   - ✅ AlertDialog открывается СРАЗУ
   - ✅ AlertDialog отображается ПОВЕРХ списка пиров
   - ✅ Фон затемнен (overlay)
   - ✅ Видно подтверждение удаления с именем пира
4. Нажать "Удалить"
5. **Ожидаемый результат:**
   - ✅ Пир удаляется с сервера
   - ✅ Список пиров обновляется
   - ✅ Показывается уведомление об успешном удалении

## Что НЕ должно происходить
❌ Модальное окно НЕ должно быть скрыто за списком пиров
❌ Модальное окно НЕ должно появляться только при возврате к списку серверов
❌ Не должно быть console.log в консоли браузера

## Демо данные
⚠️ Обратите внимание, что некоторые данные являются демо:
- График трафика: демо данные (для реальных нужно хранить историю в БД)
- Геолокация: демо координаты Москвы (для реальных нужно интегрировать ip-api.com)

## Технические детали
- **z-index модальных окон**: 200 (выше всех других элементов)
- **z-index overlay**: 50 (по умолчанию из компонентов Dialog/AlertDialog)
- **Библиотеки**:
  - `recharts` - графики трафика
  - `react-leaflet` - карта OpenStreetMap
  - `leaflet` - библиотека карт
  - `@radix-ui/react-dialog` - модальные окна
  - `@radix-ui/react-alert-dialog` - диалоги подтверждения
