## Изменения в коде

### 1. Переименование вкладки
**Файл:** `src/pages/Admin.tsx`

**Было:**
```typescript

{ id: 'blog' as AdminTab, label: 'Блог', icon: FileText },
```

**Стало:**
```typescript
{ id: 'content' as AdminTab, label: 'Управление контентом', icon: FileText },
```

### 2. Новая структура ContentTab
**Файл:** `src/components/admin/ContentTab.tsx`

Теперь это контейнер с подразделами:
- **Главная страница** - управление блоками главной страницы
- **Блог** - управление статьями (заглушка)
- **Страницы** - управление статическими страницами (заглушка)

```typescript
type ContentSubTab = 'homepage' | 'blog' | 'pages';

export function ContentTab() {
  const [activeSubTab, setActiveSubTab] = useState<ContentSubTab>('homepage');
  
  const subTabs = [
    { id: 'homepage', label: 'Главная страница', icon: Layout },
    { id: 'blog', label: 'Блог', icon: BookOpen },
    { id: 'pages', label: 'Страницы', icon: FileCode },
  ];
  
  return (
    <div>
      {/* Sub-navigation */}
      <div className="flex items-center gap-2">
        {subTabs.map(tab => (
          <button onClick={() => setActiveSubTab(tab.id)}>
            {tab.label}
          </button>
        ))}
      </div>
      
      {/* Content */}
      {activeSubTab === 'homepage' && <HomePageContentTab />}
      {activeSubTab === 'blog' && <BlogPlaceholder />}
      {activeSubTab === 'pages' && <PagesPlaceholder />}
    </div>
  );
}
```

### 3. Вынесение управления главной страницей
**Файл:** `src/components/admin/content/HomePageContentTab.tsx`

Весь функционал управления блоками главной страницы (drag-and-drop, включение/отключение) перенесен в отдельный компонент.
