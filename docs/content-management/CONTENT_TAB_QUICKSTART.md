# Управление контентом - Быстрый старт

## Что изменилось?

Раздел **"Контент сайта"** переименован в **"Управление контентом"** и теперь содержит подразделы:

```
Управление контентом
├── 📄 Главная страница  ← Управление блоками (drag-and-drop)
├── 📝 Блог              ← Управление статьями (скоро)
└── 📋 Страницы          ← Статические страницы (скоро)
```

## Как использовать

### 1. Откройте админку
```
http://localhost:8080/admin
```

### 2. Войдите
- Email: `admin@3wg.ru`
- Password: `admin123`

### 3. Перейдите в "Управление контентом"
Вы увидите три кнопки-подраздела вверху страницы.

### 4. Выберите подраздел

#### 📄 Главная страница (активно)
Управление блоками главной страницы:
- ✅ Включение/отключение блоков (кнопка 👁)
- ✅ Изменение порядка (drag-and-drop за иконку ⋮⋮)
- ✅ Сохранение изменений
- ✅ Отмена изменений

**Как использовать:**
1. Нажмите 👁 чтобы включить/отключить блок
2. Перетащите блок за ⋮⋮ чтобы изменить порядок
3. Нажмите "Сохранить"
4. Обновите главную страницу (F5)

#### 📝 Блог (в разработке)
Скоро здесь будет:
- Список статей
- Создание/редактирование статей
- Категории и теги
- SEO настройки

#### 📋 Страницы (в разработке)
Скоро здесь будет:
- Список страниц
- Создание/редактирование страниц
- Кастомные URL
- SEO настройки

## Примеры

### Пример 1: Отключить блок FAQ
1. Откройте "Управление контентом" → "Главная страница"
2. Найдите блок "FAQ"
3. Нажмите на иконку 👁 (глаз)
4. Блок станет серым
5. Нажмите "Сохранить"
6. Откройте главную страницу - блок FAQ исчез

### Пример 2: Поставить "Тарифы" выше
1. Откройте "Управление контентом" → "Главная страницу"
2. Найдите блок "Тарифы"
3. Наведите на иконку ⋮⋮ слева
4. Зажмите левую кнопку мыши
5. Перетащите блок выше
6. Отпустите кнопку
7. Нажмите "Сохранить"
8. Откройте главную страницу - порядок изменился

### Пример 3: Отменить изменения
1. Сделайте любые изменения (не сохраняя)
2. Нажмите кнопку "Отменить"
3. Все вернется к сохраненному состоянию

## Важно

### Hero секция всегда первая
Блок "Hero секция" (главный баннер) **нельзя перемещать** - он всегда остается первым.

### Сохранение обязательно
Не забывайте нажимать кнопку **"Сохранить"** после изменений!

### Обновление страницы
После сохранения обновите главную страницу (F5) чтобы увидеть изменения.

## Горячие клавиши

- **Tab** - Переключение между блоками
- **Space** - Поднять/опустить блок
- **↑↓** - Перемещение блока
- **Esc** - Отменить перемещение
