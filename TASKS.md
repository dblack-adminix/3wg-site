- [x] Fallback логика на прямые запросы при пустом кэше
- [x] Убран блокирующий спиннер при загрузке
- [x] Добавлена скелетон-анимация для плавной загрузки
- [x] Автообновление данных в фоне (без спиннеров)
- [x] Первая синхронизация сразу при старте сервера

**Результат:**
- ✅ Мгновенная загрузка страницы (< 1 секунда)
- ✅ Нет блокирующих спиннеров
- ✅ Актуальные данные (метрики каждые 2 сек, данные каждые 30 сек)

- ✅ Автообновление в фоне
